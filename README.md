
# Telecom Churn Classification & Market Basket Analysis

## Overview

This project is a Python-based data mining application developed with a Streamlit graphical user interface (GUI). The application integrates two analytical modules: Telecom Churn Classification and Market Basket Analysis, enabling users to gain insights into customer behavior and purchasing patterns through an interactive interface.

## Features

### Telecom Churn Classification

* Predicts whether a customer is likely to leave a telecom service.
* Performs data preprocessing and feature analysis.
* Utilizes machine learning techniques for customer churn prediction.
* Helps identify customers at risk of attrition.

### Market Basket Analysis

* Analyzes customer purchasing behavior using a grocery transactions dataset.
* Discovers relationships between frequently purchased items.
* Generates association rules and item recommendations.
* Provides insights for cross-selling and product placement strategies.

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Mlxtend
* Matplotlib

## Datasets

* Telecom Customer Churn Dataset
* Grocery Transactions Dataset

## Objective

The objective of this project is to apply data mining and machine learning techniques to solve real-world business problems by predicting customer churn and identifying purchasing patterns through market basket analysis.

## Application Interface

The application is hosted locally using Streamlit and provides an interactive dashboard for data analysis, visualization, and prediction.
